키보드로 아이폰 제어

  • VO Shortcuts

현재 iPhone6를 사용중이다. 원래 6s를 사용했는데 잃어버리고 중고로 얻은 폰인데 베터리가 심하게 빨리 없어진다. 완충후 출근하면 60%가 남아있다. 하루종일 충전을 하고 있어야 한다. 게다가 화면 오른쪽에 세로방향으로 4갈래 라인이 생겼는데 그 부분은 터치가 되지 않는다. 가로, 세로모드 그리고 키보드를 좌측, 우측으로 옮겨가며 정말 힘겹게 사용하고 있다.

메시지를 주고받을때 타이핑하기가 굉장히 불편한데 그래서 대안으로 블루투스 키보드를 사용한다. 그런데 사용하다보니 불편한점이 있었다. 앱의 다른버튼을 선택하거나 화면을 전환하거나 할때 키보드만으로는 제어가 안되서 손가락을 이용해 터치가 필요했던 것이다. 아이폰이 마우스까지 지원해주면 좋겠지만 마우스는 지원하지 않는다.1

키보드만으로 아이폰을 완전히 제어할 수는 없을까 알아보던중 VoiceOver기능을 알게됐다. VoiceOver기능은 시각장애인을 위한 기능으로 사용자의 모든 동작에 대해 음성으로 안내해주는 기능이다. 이 기능을 활성화하면 화면영역을 키보드로 제어할 수 있게 된다.

화면을 제어하기위해 단축키가 필요하다. 이 단축키(VO)는 Ctrl+Opt키나 Caps Lock키 조합으로 사용할 수 있으며 아이폰의 Settings > General > Accessibility > VoiceOver > Modifier Key로 조합키를 변경할 수 있다.

VO Shortcuts

자주 사용하는 키조합을 정리했다. 전체 단축키는 Use VoiceOver with an Apple Wireless Keyboard을 참조.

기본 단축키

Shortcut Description
VO+S 말하기 on/off
VO+Space Double Tab으로 앱을 실행할 때 사용
VO+H Home Button
Opt+좌우방향키 홈스크린 페이지 이동
CMD+Space 검색

빠른 탐색 단축키

아이템 이동시 VO단축키와 함께 써야하는데 빠른 탐색 모드를 사용하면 VO단축키 없이 방향키만으로 선택할 수 있다.

Shortcut Description
좌+우방향키 빠른 탐색 모드 on/off
Ctrl+위쪽 방향키, Ctrl+아래쪽 방향키 가장 첫 아이템 또는 가장 마지막 아이템 선택
위+왼쪽 방향키 위+오른쪽 방향키 rotor 전환
위, 아래 방향키 rotor 동작 선택
  1. 탈옥하면 마우스도 사용가능하다. 마우스까지 사용하게 되면 휴대용컴퓨터나 다름없어서 타제품의 경쟁력이 떨어저 일부러 기능은 제한하는거 같다. 반대로 패드에서 전화가 안되듯이. [return]
